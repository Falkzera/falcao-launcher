# types — agent.md

## Propósito
Definições TypeScript de tipos compartilhados pelo frontend. Espelham os tipos serializados pelos commands Tauri (lado Rust) e mantêm o front type-safe sem precisar regenerar bindings.

## Arquivos
- `monitor.ts` — `VmStatus`, `ContainerInfo`, `MetricPoint`, `MetricSource`, `MetricBucket` e relacionados (espelha `monitor-shared` + `src-tauri/src/monitor/queries.rs`).

## Padrões
- Cada arquivo agrupa tipos por feature (não criar um único `types.ts` gigante).
- Manter sincronizado com Rust manualmente. Tauri serializa via serde em `snake_case` por padrão (não camelCase) — TS deve casar.
- Não importar de `src/types/` em arquivos Rust — esses tipos são só consumo do front.
