# lib — agent.md

## Propósito
Utilitários TypeScript puros (parsers, helpers de cálculo, formatadores) consumidos por componentes React. Sem JSX, sem efeitos — funções determinísticas testáveis isolado.

## Arquivos
- `claudeCost.ts` — cálculo de custo equivalente de sessões Claude Code (usado no `ClaudeTab`/`TokensChart`).
- `monitor.ts` — helpers + cliente Tauri do VM Monitor:
  - `monitorApi.*` — wrappers de cada `invoke()` (vmStatus, listContainers, fetchMetricSeries, fetchLogs, healthSummary, **listStacks**, **stackDetail**).
  - `useTunnel()` — hook lifecycle do SSH tunnel (open on mount, close on unmount).
  - `usePolling(fn, ms, enabled)` — hook genérico de polling com cleanup.
- `format.ts` — `fmtBytes`, `formatRelative`, `pickUptimeColor`, `pickProgressColor`.
- `metrics.ts` — `toRate` (deriva taxa MB/s de counters cumulativos).
- `cost.ts` — `forecastMonthly` (extrapola custo Hetzner pro mês corrente).

## Padrões
- Funções puras quando possível.
- Testes (quando houver) ficam ao lado com sufixo `.test.ts`.
- Não importar de `components/` — dependência só anda no sentido contrário.
