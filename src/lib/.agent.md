# lib — agent.md

## Propósito
Utilitários TypeScript puros (parsers, helpers de cálculo, formatadores) consumidos por componentes React. Sem JSX, sem efeitos — funções determinísticas testáveis isolado.

## Arquivos
- `claudeCost.ts` — cálculo de custo equivalente de sessões Claude Code (usado no `ClaudeTab`/`TokensChart`).
- `monitor.ts` — helpers do VM Monitor: parsers de bytes (`"1.2GiB"` → number), formatters de unidades, agregações pra charts.

## Padrões
- Funções puras quando possível.
- Testes (quando houver) ficam ao lado com sufixo `.test.ts`.
- Não importar de `components/` — dependência só anda no sentido contrário.
