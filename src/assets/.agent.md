# src/assets/

Assets importados via JS (`import x from "./assets/..."`). Passam pelo bundler do Vite — ganham hash no nome em build, são tree-shaken se não forem usados.

## Conteúdo atual

- **`react.svg`** — logo do React, do scaffold. **Pode deletar** — `App.tsx` não importa mais ele depois que substituímos a tela inicial pelo grid de projetos.

## Quando colocar coisas aqui

Imagens, ícones SVG, fontes, qualquer arquivo binário que vai ser referenciado por `import` no código TS/TSX. Pra arquivos referenciados por path string em runtime, usar `public/` em vez disso.
