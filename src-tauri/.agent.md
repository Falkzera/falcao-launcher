# src-tauri/

Backend nativo + config do Tauri. **Onde vive o Rust.**

## Arquivos

- **`Cargo.toml`** — manifest Rust. Deps importantes:
  - `tauri = "2"` — framework
  - `tauri-plugin-opener = "2"` — abrir URLs no navegador padrão
  - `tokio` com features `process`, `io-util`, `sync`, `macros`, `rt`, `time` — supervisor de processos async
  - `libc` — `kill(-pgid, signal)` pra matar árvore de processos
  - `regex` + `once_cell` — detecção de porta nos logs
  - `base64` — encoda favicons como data URI
  - `dirs` — encontra `~`
  - `serde` + `serde_json` — serialização de structs e parse de `package.json`

- **`Cargo.lock`** — lockfile Rust. **Não deletar.** Garante reprodutibilidade.

- **`build.rs`** — script de build que chama `tauri_build::build()`. Não tocar.

- **`tauri.conf.json`** — config principal do Tauri. Coisas importantes:
  - `productName: "falcao-launcher"` — vira o nome do binário e o WMClass da janela
  - `identifier: "com.falcao.launcher"` — bundle ID, único por sistema
  - `build.beforeDevCommand: "pnpm dev"` — Tauri sobe o Vite antes do dev
  - `build.devUrl: "http://localhost:1420"` — porta fixa do Vite
  - `app.windows[0]` — janela principal: 1200×800, min 900×600, título "Falcão Launcher"
  - `bundle.icon[]` — lista de ícones que vão pro bundle final

## Subpastas

- **`src/`** — código Rust (commands, supervisor, scanner). **É aqui que tudo acontece.**
- **`capabilities/`** — permissões dos plugins Tauri (segurança).
- **`icons/`** — ícones gerados por `pnpm tauri icon`. Usados em build-time.
- **`gen/`** — código gerado pelo Tauri em compile time. Não editar.
- **`target/`** — output do Cargo. `target/release/falcao-launcher` é o binário final. Gigante (1-2GB), não commitar.

## Editando configs

- Mudar título da janela / tamanho → `tauri.conf.json`
- Adicionar nova permissão → `capabilities/default.json`
- Trocar ícone → fonte é `../icon-source.svg`, regerar via `pnpm tauri icon`
- Adicionar dep Rust → `Cargo.toml` + rodar `cargo check`

## Recompilar manualmente

Tauri dev recompila ao detectar mudança em `src-tauri/src/*`. Pra forçar (ex: trocou um arquivo em `icons/`):

```bash
touch src-tauri/src/lib.rs
```
